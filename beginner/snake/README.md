# Snake Game 🐍

A classic Snake Game built using Python and Pygame as part of the `PythonForAgenticAI` learning journey.

---

# 🚀 Features

- Snake movement using arrow keys
- Random food generation
- Snake body growth
- Score tracking
- Wall collision detection
- Self collision detection
- Game Over screen
- Restart game support

---

# 🛠 Technologies Used

- Python
- Pygame

---

# 📦 Installation

## Clone Repository

```bash
git clone <your-github-repo-url>
```

## Navigate to Project

```bash
cd PythonForAgenticAI
```

## Activate Virtual Environment

### Mac/Linux

```bash
source .venv/bin/activate
```

### Windows

```bash
.venv\Scripts\activate
```

## Install Dependencies

```bash
pip install pygame
```

---

# ▶️ Run the Game

```bash
python beginner/snake/snake.py
```

or

```bash
python3 beginner/snake/snake.py
```

---

# 🎮 Controls

| Key | Action |
|---|---|
| ⬅️ | Move Left |
| ➡️ | Move Right |
| ⬆️ | Move Up |
| ⬇️ | Move Down |
| C | Play Again |
| Q | Quit Game |
